注明CPU是AMD的！虚拟机是VIRTUAL BOX

第一步，先查看自己电脑的CPU支不支持虚拟化（工具Securable，自己下一个） 理论上不支持虚拟化的CPU是不能装的

 

当测试结果显示为“Hardware Virtualization YES”或者“Hardware Virtualization Locked On”之类的字眼就证明已经开启了CPU的虚拟技术支持了。我使用的是华硕m4a88t-m主板，bois中需要将 Advanced/Secure Virtual Machine Mode 的值设置为Enabled （默认是Disabled）。 （纠正一下，上面的图片个这段话是引用别人的）

 

第二步

下载地址

 

雪豹系统：http://www.verycd.com/topics/2765539/

     雪豹更新补丁：http://support.apple.com/kb/dl1349

     关键的一个东西：http://ishare.iask.sina.com.cn/f/13524370.html?retcode=0 （legacy_kernel-10.6.0.pkg）

Ps：这个补丁可以直接在MAC里面下载~或者用U盘拷进去

     XCODE4.1：ed2k://|file|[寮€鍙戝伐鍏穄.TLF-SOFT-Apple.Xcode.v4.0.1.and.iOS.SDK.v4.3.MacOSX.ISO-CORE.dmg|4607939522|c726159d07ab97c952451eea248d6267|h=3j6lfuct727aqvo54buwpympp4ccn5jd|/  这个复制了用迅雷下

 

 

第三部

建立虚拟机

新建一个MAC就OK啦

设置如图

 

 

1、移除软驱

2、启用EFI（只针对某些系统） 不要打勾

3、MAC OS X是64bit

4、其他设置无伤大雅！

 

 

第四步 安装

一开始的引导文件是 可以是 EmpireEFI4AMD.ios 也可以是 darwin_snow_legacy.ios

分配光驱那里选择 darwin_snow_legacy.ios 或 EmpireEFI4AMD.ios （不再重复了）

 

网上的说法是不同的cpu会用到不同的引导文件，经实践发现不仅cpu，安装不同版本的mac os都会有影响，一个一个试吧，总有适合的。

设置完成后运行mac虚拟机，如果引导文件正确，就会看到这个（当然不同的引导文件的画面不一样）：

 

 

 

 

 

 

 

 

设备 分配光驱 去分配  [苹果雪豹操作系统正式版].snowleopard_10a432_userdvd.dmg

这个家伙 dmg 后缀 没所谓。

dmg文件选择后，按 F5 键后，界面中的Legacy EmpireEFI 图标将变为 Mac OS X Install DVD 图标,如下：

 

 

 

PS：有些图我直接拷贝网上别人的 你自己看实际情况！

此时按回车键，过会儿就可以进入系统的安装界面了。

 

 

这个画面可能有点久 慢慢来！

 

 

 

 

 

 

 

虽然提示安装失败，但其实已经成功安装完毕了。

这个时候用darwin_snow_legacy.ios 或 EmpireEFI4AMD.ios 也是可以启动的！

 

 

至此Mac OS X基础系统安装完毕！

 

第五步

安装系统更新+补丁+ XCODE

安装完系统后4.1的XCODE没法安装，所以我们必须更新系统，

 PS：系统直接更新 我还未测试好！

用我们之前下载的更新

仍然是分配光驱 强制释放推出光盘的选项

然后安装~~一路点下去~~~~~、

 

【升级完成以后系统会叫你点Restart（重启）。千万千万不要马上点，点了后重启鼠标就不能动了，又要重装~~（我就在这搞了两天，不信你们可以试试）】

PS：上面那句话摘自网上攻略 我未测试过 不确定！

不过我还是装了

legacy_kernel-10.6.0.pkg 补丁

 

然后重启 要开始换引导了！ 我测试了很多个引导文件 貌似只有darwin300 可以用！

（而且方法很雷人 就是分配 然后取消分配 分配取消分配 我怀疑他的启动是随机的！）

 

取消分配光驱

就可以进入系统了！

 

Ps：关机每次都要强制关机才能关掉这个虚拟机，开机每次都得分配光驱 取消光驱这个方式去开。

 

关于XCODE安装~

安装XCODE4.1

修改日期为安装包附近的日期改成2011年6月，也就是ios4.3发布后不久，也就可以装了

PS:记得关掉网络 不然MAC会自动更新时间 然后程序就会突然出错！！

 

最新的Xcode4.3.2 and IOS SDK 5  需要10.7.3以上版本
