--- layout: post title: '''/bin/bash --login'' command problem' categories: - RubyOnRails - ubuntu tags: - it status: publish type: post published: true meta: duoshuo_thread_id: '1365771562581491733' _edit_last: '1' --- If you run "rvm use XXX" then output "'/bin/bash --login' as the command"
You can do as follow:
1. Open a terminal,
2. Choice the drop-down menu 'Edit'
3. Click the menu 'Profile Preferences'

4. Choice 'Title and Command'
5. Click the command 'Run command as a login shell'
Then you can't run '/bin/bash --login' next time.
Reference:
Integrating RVM with gnome-terminal
https://rvm.io/integration/gnome-terminal
